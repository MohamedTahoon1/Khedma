# 🏛️ Public Services Mobile App

A modern mobile application that provides easy access to essential government and public services. The app focuses on simplicity, clarity, and fast navigation to help users complete official requests efficiently.

---

## 📱 Features

- 🏠 **Home Dashboard**
  - Centralized access to all services
  - Clean and minimal interface
  - Personalized greeting

- 🔍 **Search Services**
  - Quickly search for available services
  - Easy-to-use search bar

- ⭐ **Popular Services**
  - ID Renewal
  - Birth Certificate Request
  - Passport Application / Renewal
  - Marriage Certificate Request

- 🔔 **Notifications**
  - Bell icon for service updates and alerts

- 📂 **Document Management**
  - My Documents section
  - Saved services and requests

- 👤 **User Profile**
  - View and manage personal information
  - Account settings

- 🧭 **Bottom Navigation Bar**
  - Home
  - My Docs
  - Saved
  - Profile
